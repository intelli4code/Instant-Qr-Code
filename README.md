**Instant QR** is a high-authority, professional-grade QR code generation toolkit designed for modern businesses. Built with a focus on premium aesthetics, client-side privacy, and powerful utility features.

---

## ✨ Key Features

### 🖼️ Branding & Customization
- **Logo Integration**: Upload `.png` or `.svg` branding to place in the center of your QR codes.
- **Precision Design**: Customize foreground/background colors, dot patterns (squares, dots, rounded), and frame styles.
- **Real-time Preview**: See changes instantly with our high-resolution rendering engine.

### 🗄️ Advanced Utility
- **Local History**: Automatically saves your last 10 unique configurations locally for easy retrieval.
- **PDF Print Templates**: Export print-ready 3x4 label grids for physical distribution.
- **Bulk Generator**: Upload a CSV file to generate hundreds of QR codes at once, exported as a tidy ZIP archive.

### 📱 User Experience
- **Scan Simulator**: A high-fidelity mobile mockup that lets you "pre-scan" your QR code to verify its appearance.
- **Light/Dark Mode**: A persistent, sleek Deep Navy theme (Dark) and a clean, corporate White theme (Light).
- **Responsive Layout**: Optimized for every device, from ultra-wide monitors to mobile smartphones.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/intelli4code/Instant-Qr-Code.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
